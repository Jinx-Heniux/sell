# Spring Boot开发的微信点餐系统

## 按用户角色划分功能模块:
### 买家（手机端):
- 下单
- 查询订单
- 取消订单
- 支付订单

### 卖家（PC端）:
- 类目增删
- 商品上下架
- 接单
- 查询订单
- 取消订单

## 使用到的技术:
### Spring Boot
- SpringBoot + JPA
- SpringBoot + Mybatis
- SpringBoot + Redis （待完成）
- SpringBoot + WebSocket（待完成）

### Wechat
- 微信授权登录
- 模板消息推送（待完成）
- 支付与退款

### Redis（待完成）
- Jedis的相关操作
- 分布式Session
